公開のやり方（完全無料）

  1. このファイルを保存：名前は index.html
  2. GitHubに新規リポジトリを作成（公開/Public）し、index.html をアップロード
  3. Settings → Pages で Deploy from a branch を選び、main / root を指定 → Save
  4. 数十秒後、表示された URL にアクセスすれば公開完了！

他にも Cloudflare Pages や Netlify Drop に index.html をドラッグ＆ドロップでも一瞬で公開できます。
